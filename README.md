# Fichas D&D

## Descrição

Site interativo para criação e gerenciamento de fichas de personagens inspiradas em Dungeons & Dragons.

O projeto terá foco em:

* Interface temática
* Criação simplificada de personagens
* Organização visual da ficha
* Interatividade
* Experiência do usuário

---

# Identidade Inicial

## Estilo visual

* Dark fantasy
* Medieval
* Interface inspirada em RPGs

## Sensação desejada

O site deve transmitir a sensação de estar utilizando uma ficha real de RPG em um sistema moderno e estilizado.

---

# Objetivo do Projeto

Desenvolver um site interativo para criação e visualização de fichas de personagens inspiradas em Dungeons & Dragons.

O foco principal do projeto será:

* Interface visual imersiva
* Facilidade de criação de personagens
* Interatividade
* Organização das informações da ficha

## Limitações da versão atual

O projeto NÃO terá:

* Combate online
* Multiplayer
* Sistema completo de campanha

---

# Público-alvo

* Jogadores iniciantes de RPG
* Pessoas interessadas em D&D
* Visitantes da feira de tecnologia
* Colegas e professores

---

# Funcionalidades Principais (MVP)

## 1. Criação de personagem

O usuário poderá:

* Escolher nome
* Escolher raça
* Escolher classe
* Escolher alinhamento
* Escolher nível

---

## 2. Sistema de atributos

A ficha terá:

* Força
* Destreza
* Constituição
* Inteligência
* Sabedoria
* Carisma

### Funcionalidades

* Inserção manual de valores
* Rolagem automática de atributos

---

## 3. Cálculo automático

O sistema calculará automaticamente:

* Modificadores de atributo
* Vida básica
* Bônus de proficiência

---

## 4. Exibição visual da ficha

A ficha mostrará:

* Avatar do personagem
* Informações básicas
* Atributos
* Equipamentos
* Magias (opcional)
* História/personalidade

---

## 5. Salvamento local

As fichas poderão ser:

* Salvas no navegador usando LocalStorage
* Reabertas posteriormente
